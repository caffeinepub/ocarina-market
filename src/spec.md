# Specification

## Summary
**Goal:** Replace itinerary management with enhanced item display, showing quantity and organizing items by categories and subcategories.

**Planned changes:**
- Remove all itinerary-related code from backend (types, CRUD operations)
- Remove all itinerary-related frontend pages, components, routes, and hooks
- Add quantity field display to ItemCard component
- Implement item sorting by category (Ceramic vs 3D Printed) and subcategory (shape)

**User-visible outcome:** The storefront displays items organized by category and subcategory with visible quantity information. Itinerary features are completely removed from the application.
